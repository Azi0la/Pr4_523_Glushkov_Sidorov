## Функциональные возможности

### Главная Страница
* Кнопки перехода на примеры (Функция 1, Функция 2, Функция 3)

### Страница с первой функцией
* Ввод: x, y, z
* Отображение скриншота формулы из вышеупомянутой таблицы (№1)
* Кнопка "Вычислить" - расчёт функции с помощью введенных значений и их вывод в поле ответа
* Кнопка "Очистить" - очищение полей ввода и ответа
* Кнопка "Назад" - возврат на главную страницу

### Страница со второй функцией
* Ввод: x, y
* Отображение скриншота формулы из вышеупомянутой таблицы (№2)
* 3 кнопки переключателя для выбора функции f(x)
  * sh(x)
  * x<sup>2<sup/>
  * e<sup>x<sup/>
* Кнопка "Вычислить" - расчёт функции с помощью введенных значений, выбора функций и их вывод в поле ответа
* Кнопка "Очистить" - очищение полей ввода и ответа
* Кнопка "Назад" - возврат на главную страницу

### Страница с третьей функцией
* Ввод: x<sub>0</sub>, x<sub>k</sub>, dx, a, b
* Отображение скриншота формулы из вышеупомянутой таблицы (№3)
* Кнопка "Вычислить" - вычисление значений y(x) на отрезке ( x<sub>0</sub> - x<sub>k</sub> ) , а также их вывод в поле ответа и построение графика функции
* Кнопка "Очистить" - очищение полей ввода и ответа
* Кнопка "Назад" - возврат на главную страницу
